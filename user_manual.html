1. Overview
📊 What is VilfinTV Multiscreener?
VilfinTV Multiscreener is a professional, browser-based financial research tool designed for multi-asset analysis across stocks, ETFs, mutual funds, bonds, commodities, currencies, and more. It combines an AI-powered research input form, live market data via TradingView, financial calculators, global news aggregation, and optional Cloudflare Workers integration for a serverless Claude AI backend — all in a single self-contained HTML file, no installation required.
🗺️ Layout at a Glance
Top Menu Bar — sticky header with File, News, Financial News, Financial Video, Tools menus + VilfinTV brand + Cloudflare status.
Left Sidebar (icons) — quick access to Instructions, Asset Downloads, Brokers, Insurance, NRE FD, Tax, Loans, and more.
Main Content Area — three tabs: Input Form, Copy AI Query, Paste & Render.
Right Bar (Live Markets) — always-visible TradingView live widgets: Overview, Indices, Stocks, Commodities, Forex.

3. File Menu & Cloudflare Workers
🔗 Cloudflare Workers — Connect
Cloudflare Workers is a serverless platform where you can host a small API that stores your Claude API key securely — so you don't need to paste the key into the browser each session.
1
Deploy a Cloudflare Worker with your Claude API key stored as a secret environment variable.
2
Click File → Cloudflare Workers → Connect. A modal dialog will appear.
3
Enter your Worker URL (e.g. https://my-worker.workers.dev) and click Connect.
4
The URL is saved to your browser's localStorage — no server storage. On the next visit, it reconnects automatically.
5
To disconnect: File → Cloudflare Workers → Disconnect. The saved URL is removed from localStorage.
Privacy: The Worker URL is stored only in your local browser (localStorage). It is never sent to any third party.
✕ Exit
Clicking File → Exit will prompt for confirmation and attempt to close the browser tab/window.

10. Input Form
1
Q1 — Asset Type: Select from Stock, ETF, Mutual Fund, Bond, Commodity, or Currency.
2
Q1a — Market: Select the relevant market (India, Global, US, Japan etc.) based on Q1.
3
Q1b/Q1c — Sub-type: Additional sub-classifications appear where relevant (e.g. bond type, commodity sub-type).
4
Q2 — Asset Name: Type the name or ticker of the asset you want to research.
5
Q3 — Resident Status: Resident Indian, NRI, or Japan Resident. This tailors tax & regulatory information.
6
Q4 — Investment Horizon: Short, Medium, or Long term.
7
Q5 — Amount (optional): Select currency and enter an investment amount for personalised allocation advice.
8
Click the orange Generate AI Query button. You will be automatically taken to the Copy AI Query tab.
11. Copy AI Query
🤖 Using the Generated Prompt
Click Copy Full Query to copy the complete research prompt to your clipboard. Then:

1. Open claude.ai (or your preferred AI tool).
2. Paste the query and send it.
3. The AI will return structured research data (not HTML code) — this is intentional and is fast.
4. Copy the entire AI response.
Note: If you have a Cloudflare Worker connected, the Worker can be extended to call the Claude API directly. The Worker URL is available in window._cfWorkerUrl for custom integrations.
12. Paste & Render
1
Switch to the Paste & Render tab.
2
Paste the AI response into the large text area.
3
Click Generate Report. The full research report renders instantly in the browser.
4
Use Export as HTML to save a standalone file, or Export as PDF to print / save as PDF.
If any data field is missing, the report shows "No Data Available" rather than leaving it blank — this makes gaps visible.
13. Financial Calculators
📈 SIP Calculator
Calculate returns for a Systematic Investment Plan with optional Step-Up SIP. Supports INR, JPY, USD, EUR, GBP, SGD.
💰 Lumpsum Calculator
Calculate the future value of a one-time lump sum investment at a given annual return rate.
🔢 SIP + Lumpsum
Combines a monthly SIP with an initial lump sum. Shows total corpus, wealth gained, and year-by-year growth.
🥧 Asset Allocation
Enter your age, risk profile, investment horizon, and amount to get a personalised asset allocation across Equity MF, Stocks, Hybrid, Bonds, Gold, Silver, REITs, NRE FD. Insurance premiums are factored in as real annual costs.
⚖️ Compare Funds & ETF
Enter two fund names and generate a side-by-side AI comparison query. Paste the AI output to render a comparison report.
14. Tips & Shortcuts
💡 Useful Tips
• Digital Gold/Silver: Select Commodity → Metals → Gold/Silver → Digital sub-option for a specialised report.

• Mutual Fund FOF: Enter the fund name — the AI query is configured to also pull underlying ETF lists and expense ratios.

• Escape key: Closes any open left sidebar overlay panel.

• Right bar tabs: Switch instantly between Overview, Indices, Stocks, Commodities, Forex — no page reload.

• Cloudflare Worker URL is remembered across browser sessions via localStorage. Clear your browser storage to reset it.

• Print/PDF: The rendered research report is print-optimised. Use Export as PDF for a professional output.

• Mobile: On screens under 1100px the right bar is hidden. Use Tools → Live Markets to access widget tabs.
Data Disclaimer: All market data in the Live Markets bar is provided by TradingView (free widgets) and is subject to TradingView's terms. Report data is AI-generated from publicly available information — not investment advice.
